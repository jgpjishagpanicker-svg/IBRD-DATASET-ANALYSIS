# IBRD-DATASET-ANALYSIS
The objective is to understand status of different types of loans disbursed to countries under different interest rates for several projects and their repayment patterns.
An end-to-end data analytics workflow on the International Bank for Reconstruction and Development (IBRD) loan dataset using Python in Jupyter Notebook.
Used Python ibraries such as Pandas,Matplotlib,Plotly and Seaborn.
